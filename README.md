# KR 얼리버드 오가닉 생성기

Figma에서 오가닉 가이드 4개 시안을 한 번에 생성하는 플러그인입니다.

## 플러그인 사용 대상

- 실제로 시안을 생성하는 디자이너
- 사내 공유 폴더 또는 Git 저장소에서 플러그인을 받아 사용하는 팀원

## 개요

- 플러그인 내부에 내장된 4개 템플릿을 현재 페이지에 바로 생성합니다.
- 각 시안은 부모 프레임 없이 독립 Frame으로 가로 배치됩니다.
- 입력값은 `job_title`, `speaker_name`, `d_day`, `img` 레이어 기준으로 반영됩니다.
- 이미지 입력은 공개 구글 드라이브 파일 링크 또는 파일 업로드 중 하나만 사용할 수 있습니다.

## 입력 규칙

- 직업명: 필수
- 연사명: 필수
- D-day 숫자: 필수, 한 자리 숫자만 입력 가능
- 이미지: 공개 구글 드라이브 파일 링크 또는 업로드 파일 중 하나 필수

## 생성 규칙

- 선택한 노드가 있으면 해당 노드 위치 기준으로 생성합니다.
- 선택한 노드가 없으면 현재 보이는 캔버스 영역 시작점 기준으로 생성합니다.
- 4개 시안은 일정 간격으로 가로 정렬됩니다.
- 텍스트는 Spoqa Han Sans Neo를 사용합니다.

## 설치 방법

1. 사내 공유 폴더 또는 사내 Git 저장소에서 플러그인 폴더를 받습니다.
2. Figma Desktop을 엽니다.
3. `Plugins > Development > Import plugin from manifest...`를 선택합니다.
4. 받은 폴더 안의 `manifest.json` 파일을 선택합니다.
5. `Plugins > Development > KR 얼리버드 오가닉 생성기`로 실행합니다.

## 사용 방법

1. 플러그인을 실행합니다.
2. `직업명`, `연사명`, `D-day 숫자`, `이미지`를 입력합니다.
3. `Generate`를 누릅니다.
4. 현재 페이지에 4개 시안이 가로로 생성됩니다.

## 이미지 입력 안내

- 공개 구글 드라이브는 폴더 링크가 아니라 이미지 파일 1개의 공개 링크를 사용해야 합니다.
- 파일 업로드와 구글 드라이브 링크는 동시에 사용할 수 없습니다.

## 오류 안내

- 플러그인 실행 중 오류가 나면 디자인팀에 문의해주세요.
