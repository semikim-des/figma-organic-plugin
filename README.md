# KR 얼리버드 오가닉 생성기

Figma에서 오가닉 가이드 4개 시안을 한 번에 생성하는 플러그인입니다.

## 개요

- 플러그인 내부에 내장된 4개 템플릿을 현재 페이지에 바로 생성합니다.
- 각 시안은 부모 프레임 없이 독립 Frame으로 가로 배치됩니다.
- 입력값은 `job_title`, `speaker_name`, `d_day`, `img` 레이어 기준으로 반영됩니다.
- 이미지 입력은 공개 구글 드라이브 파일 링크 또는 파일 업로드 중 하나만 사용할 수 있습니다.

## 입력 규칙

- 직업명: 필수
- 연사명: 필수
- 디데이 숫자: 필수, 한 자리 숫자만 입력 가능
- 이미지: 공개 구글 드라이브 파일 링크 또는 업로드 파일 중 하나 필수

## 생성 규칙

- 선택한 노드가 있으면 해당 노드 위치 기준으로 생성합니다.
- 선택한 노드가 없으면 현재 보이는 캔버스 영역 시작점 기준으로 생성합니다.
- 4개 시안은 일정 간격으로 가로 정렬됩니다.
- 텍스트는 Spoqa Han Sans Neo를 사용합니다.

## 파일 구성

- `manifest.json`: Figma 플러그인 설정 파일
- `code.js`: 플러그인 메인 실행 코드
- `ui.html`: 플러그인 UI 및 입력 검증 로직
- `merged-react-figma.json`: 이번 구현 기준 템플릿 구조 참고용 JSON
- `package.json`: 프로젝트 기본 메타 정보
- `eslint.config.js`: 린트 설정 파일

## 실제 수정 포인트

- 시안 구조/생성 로직 수정: `code.js`
- UI 문구/스타일/입력 검증 수정: `ui.html`
- 참고용 템플릿 구조 확인: `merged-react-figma.json`
- 플러그인 메타 정보 수정: `manifest.json`

## 실행 방법

1. Figma Desktop에서 `Plugins > Development > Import plugin from manifest...`를 선택합니다.
2. 이 폴더의 `manifest.json`을 불러옵니다.
3. 플러그인을 실행합니다.
4. 입력값을 넣고 `Generate`를 누르면 시안 4개가 생성됩니다.

## 참고 사항

- 현재 실행 파일은 `code.js`입니다.
- 템플릿 데이터는 실행 시 `code.js`와 `ui.html` 내부 값을 사용합니다.
- `merged-react-figma.json`은 유지보수용 참고 파일입니다.
- 구글 드라이브는 폴더 링크가 아니라 이미지 파일 1개의 공개 링크를 사용해야 합니다.
